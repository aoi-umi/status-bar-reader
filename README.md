# status bar reader

## This is the README for your extension pack "status-bar-reader"

you can read book on the status bar

| Key          | Description   |
| :----------- | :------------ |
| `ctrl+alt+=` | active/toggle |
| `ctrl+right` | next          |
| `ctrl+left`  | prev          |
| `ctrl+down`  | nextLine      |
| `ctrl+up`    | prevLine      |
| `ctrl+alt+l` | go to line    |
